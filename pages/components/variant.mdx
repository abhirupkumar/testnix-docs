# Variant

The ```<Variant />``` component encapsulates the content of each variant in your experiment. A ```<Variant />``` can have either a server or a client component.

## Props

- `variantId` `string{:sh}` `required{:cs}`

<p className="mt-4 ml-8">Any unique Id can be provided.</p>

- `children` `JSX Elements{:sh}` `required{:cs}`

<p className="mt-4 ml-8">Any valid Jsx elements.</p>
